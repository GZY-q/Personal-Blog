# ✅ **个人博客 / 作品集网站生成 Prompt（可直接复制）**

**角色描述：**
你是一名资深前端设计师 + 全栈架构师 + UI/UX 专家，请严格按以下要求生成完整的个人博客/作品集网站页面，包括：布局、样式、组件结构、文案与交互设计。

---

# 🎨 **【网站类型】**

一个干净、专业、现代的 **个人博客 + 作品集网站**，用于展示：

* 个人简介（About）
* 技能（Skills）
* 项目经历（Projects）
* 博客文章列表（Blog）
* 联系方式（Contact）
* 可下载的 PDF 简历链接

主题风格简洁现代，偏科技风，适合软件工程专业毕业生。

---

# 📐 **【设计风格要求】**

* 整体风格：简洁、科技、清爽
* 色调：浅色主题（白色 + 灰色 + 科技蓝）
* 布局：响应式（PC/Pad/手机）
* 字体：无衬线，现代感
* 动效：轻微淡入、滑动动画
* 导航栏固定在顶部，向下滚动时有阴影
* 页面内容使用 1200px 居中布局

---

# 🧱 **【页面结构要求】**

## 1. Header（导航栏）

* Logo（文字即可）
* 导航菜单：About / Skills / Projects / Blog / Contact / Resume

## 2. Hero（首页横幅）

* 欢迎标语：
  **“Hi, I'm GZY — A Software Engineer & Web Developer.”**
* 一段简短介绍（自动生成，但要保持专业）
* 一个 CTA 按钮（View Projects / Download Resume）
* 一张科技风插画/头像占位图

## 3. About Me（个人简介）

包含：

* 我是谁
* 专业：软件工程
* 方向：Web开发 / 全栈 / Node.js / 前端工程化 / AI 工具
* 目标：找工作 + 做个人项目
* 一张侧边照片或插画

## 4. Skills（技能）

图标 + 标签形式
**分类：**

* Frontend: HTML, CSS, JavaScript, React, Vue（自动生成）
* Backend: Node.js, Express, MongoDB
* Tools: Git, GitHub, Docker, Linux, VSCode
* Others: REST API, WebSocket, AI 工具

## 5. Projects（项目）

现代卡片风格，每项包括：

* 项目标题
* 项目描述
* 技术栈标签
* 项目截图占位图
* GitHub 链接按钮
* 在线 Demo 按钮（如无可隐藏）

至少展示 **3 个项目**，描述真实、专业。

## 6. Blog（博客列表）

* 文章卡片（标题、简介、发布日期）
* AI 自动生成 3–5 篇示例文章：

  * “如何使用 Node.js 构建多人联机小游戏”
  * “我的软件工程毕业项目总结”
  * “前端工程化最佳实践（2025）”

## 7. Contact（联系方式）

包含：

* 邮箱（占位）
* GitHub 链接
* LinkedIn 链接
* Contact Me 按钮（邮件表单）

## 8. Footer

* 版权
* “Made by GZY”
* GitHub 图标链接
